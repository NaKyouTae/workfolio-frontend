# Cursor Configuration

이 프로젝트는 `../claude-settings/.claude`의 공유 설정을 참조합니다.

## 공유 설정 사용

공유 설정은 다음 위치에 있습니다:
- `../claude-settings/.claude/rules/` - 공통 코딩 규칙
- `../claude-settings/.claude/skills/` - 재사용 가능한 스킬
- `../claude-settings/.claude/agents/` - 에이전트 설정

## 프로젝트별 설정

프로젝트별 규칙이나 설정이 필요한 경우 이 디렉터리에 추가할 수 있습니다.
